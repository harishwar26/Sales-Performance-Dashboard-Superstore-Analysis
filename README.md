# Sales-Performance-Dashboard-Superstore-Analysis

📊 Superstore Sales Dashboard Analysis

🎯 Overview

This project transforms raw retail data into an interactive business intelligence dashboard. The goal was to identify seasonal sales trends, regional performance, and product category distribution to help stakeholders make data-driven inventory and marketing decisions.

🛠️ Tech Stack

Data Visualization: Power BI (Desktop)

Data Cleaning: Python + Pandas (Optional processing)

Dataset: Superstore Sales CSV (Order Date, Region, Category, Sales, Profit)

⚡ Data Transformation Steps

Data Ingestion: Imported the raw Superstore_Sales.csv into Power BI.

Date Modeling: Used Power Query to transform the Order Date column into a Month-Year format to allow for chronological time-series analysis.

UI/UX Design: * Applied a consistent color palette to highlight top-performing metrics.

Implemented a Slicer for Region and Category to allow for dynamic "drill-down" analysis.

📈 Dashboard Components

Line Chart: Visualizes Sales trends over time to identify peak shopping seasons.

Bar Chart: Compares Sales by Region, using conditional formatting to highlight the leader.

Donut Chart: Breaks down Sales by Product Category (Technology, Furniture, Office Supplies).

💡 Key Business Insights

Peak Seasonality: Sales consistently spike in November and December, indicating a heavy reliance on Q4 holiday shopping.

Regional Leader: The West Region represents the highest sales volume, suggesting a more mature market or higher customer density.

Category Performance: While Technology generates the highest revenue, the Furniture category shows lower profit margins, likely due to logistics and shipping overhead.
